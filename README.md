# skipass-cloud
SkiPass Cloud – Statischer Demo-Webshop
Projektbeschreibung
SkiPass Cloud ist ein fiktives Software-Produkt (SaaS), das den Verkauf und die Verwaltung von Skipässen für Skigebiete digital unterstützt.
Im Rahmen dieser Übung wurde ein statischer Webshop umgesetzt, der den Aufbau und Ablauf eines realen Software-Webshops simuliert. Alle Prozesse (z. B. Checkout, Zahlung, Analytics) sind gemockt und dienen ausschließlich Demonstrationszwecken.
Das Projekt wurde als Übung für rechtliche, gestalterische und technische Aspekte von Webshops entwickelt und stellt keinen produktiven Betrieb dar.

# Softwareprodukt
Produktname: SkiPass Cloud
Produkttyp: Business-Software (Software as a Service)
Funktionen der Anwendung
•	Online-Ticketshop für Skipässe
•	Konfigurierbare Tarife und Abonnements
•	Digitaler QR-Ticket-Versand (Demo)
•	Zutrittskontrolle per QR-Code (UI-Simulation)
•	Reporting- und Auswertungsansichten (Beispieldaten)
•	Rollenmodell (Admin / Verkauf / Kontrolle)
•	Barrierefreier Webshop (WCAG 2.2 AA)

# Zielgruppe
Die Anwendung richtet sich an:
•	Skigebietsbetreiber
•	Liftgesellschaften
•	Betreiber kleiner bis großer Skianlagen
Der Fokus liegt auf B2B-Kunden, die ihre Ticket- und Zutrittsprozesse digitalisieren möchten.

# Unternehmen

# Geschäftsmodell
SkiPass Cloud wird als Software-as-a-Service (SaaS) angeboten.
•	Abonnement-Modell (monatliche Zahlung)
•	Unterschiedliche Pläne (Basic, Pro, Enterprise)
•	Preisstaffelung nach Funktionsumfang und Nutzerkonten
In dieser Übung sind Zahlung, Vertragsabschluss und Bereitstellung vollständig gemockt.

# Technische Umsetzung
•	Statische Website (HTML, CSS, JavaScript)
•	Kein Backend, keine echte Datenübertragung
•	Warenkorb und Demo-Zustände werden lokal im Browser gespeichert
•	Hosting z. B. über GitHub Pages / Cloudflare Pages
